These files contain the room designs. TMX files contain the raw tile compositions and can be opened with Tiled (https://www.mapeditor.org/). The TMX files rely on the tileset definitions that can be found in the "tilesets" folder. 
XCF files contain layering information with additional custom objects, as well as shading and lighting layers and can be opened using GIMP (https://www.gimp.org/).
